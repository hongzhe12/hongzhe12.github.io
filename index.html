好贴板—跨平台剪贴板管理与多设备同步工具

核心功能

无限历史记录管理

支持自定义最大记录数（默认10万条），自动清理旧记录，节省存储空间

智能搜索与标签系统

正则表达式精准搜索，动态标签多级分类，搜索防抖优化，减少无效查询

多设备同步与 Web 管理

局域网实时互传，手机扫码快速连接，Web 控制台支持远程查看/导出数据

技术特点

  • 前端：PySide6 原生 GUI
  • 后端：Flask + SocketIO 实时通信
  • 数据库：支持 SQLite / PostgreSQL / MySQL
